## 安全境界

- LINE の送信先は、あやこ `U5217ceb4debd9849959446ce8f902a27` **だけ**にする。他の友だち ID では実行しない。
- メールは host closer が用意したテスト用メールアドレスだけを使う。個人の通常アドレスは使わない。
- その場で作る使い捨ての internal フォームだけを使う。本番3フォーム、Formaloo フォーム、既存回答には触れない。
- sandbox/lane から LINE・メールを実射しない。査読済み revision と migration 124 を反映した trusted host で行う。
- token、API key、password、編集リンク全文、回答者の PII をログ・スクリーンショット・報告文へ貼らない。
- 受信証跡は、時刻、チャネル、フォーム ID、宛先のマスク値、結果だけを残す。
