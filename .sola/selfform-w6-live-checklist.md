# selfform-w6-submission-notify — host live checklist

## 現在の状態

- 実受信: **未実施（host closer 工程）**
- lane で完了済み: 自動テスト、migration 124、送信先の構造保証、署名付き編集リンク、文面プレビュー
- この手順の目的: LINE 経由 1 周と埋め込み経由 1 周を、trusted host で実際に受信して確かめる

## 安全境界

- LINE の送信先は、あやこ `U5217ceb4debd9849959446ce8f902a27` **だけ**にする。他の友だち ID では実行しない。
- メールは host closer が用意したテスト用メールアドレスだけを使う。個人の通常アドレスは使わない。
- その場で作る使い捨ての internal フォームだけを使う。本番3フォーム、Formaloo フォーム、既存回答には触れない。
- sandbox/lane から LINE・メールを実射しない。査読済み revision と migration 124 を反映した trusted host で行う。
- token、API key、password、編集リンク全文、回答者の PII をログ・スクリーンショット・報告文へ貼らない。
- 受信証跡は、時刻、チャネル、フォーム ID、宛先のマスク値、結果だけを残す。

## 0. 使い捨てフォームと通知文面を準備

- [ ] 高機能フォームを 1 件だけ新規作成し、配信方式を「自前フォーム」にする。
- [ ] `お名前`（1行テキスト・必須）と `確認メール`（メール・必須）を置き、公開する。
- [ ] 「回答後の自動通知」を ON にし、「回答者本人のメール項目」で `確認メール` を選ぶ。
- [ ] 文面に `{{display_name}}`、`{{回答:お名前}}`、`{{編集リンク}}` を差し込み、プレビューが実際に読める文章になることを確認する。
- [ ] 設定を保存し、再読込後も ON・メール項目・文面が残ることを確認する。

## 1. LINE 経由を 1 周

- [ ] フォームに紐づく LINE アカウントと、あやこ `U5217ceb4debd9849959446ce8f902a27` の友だちレコードが一致していることを host 側で確認する。
- [ ] W3 の LINE 用公開導線から使い捨てフォームを開く。ブラウザの URL や報告へ `fr_id` を貼らない。
- [ ] 架空の回答とテスト用メールを入力して 1 回だけ送信する。
- [ ] あやこの LINE にだけ、フォーム名、回答内容、編集リンクを含む確認通知が 1 回届くことを確認する。
- [ ] LINE の編集リンクを開き、`お名前`を別の架空名へ再編集して保存する。
- [ ] 「保存しました」が表示され、同じリンクを再読込すると変更後の値が表示されることを確認する。
- [ ] 管理画面の回答も変更後の値で、別の友だち・別フォームへ影響がないことを確認する。

## 2. 埋め込み経由を 1 周

- [ ] 新しいシークレット/プライベートブラウザで通常の埋め込み公開 URL を開き、LINE 用パラメータがないことを確認する。
- [ ] `確認メール`へテスト用メールアドレスを入力し、架空の回答を 1 回だけ送信する。
- [ ] そのテスト用メールだけに、件名、フォーム名、回答内容、編集リンクを含む確認メールが 1 通届くことを確認する。
- [ ] メールの編集リンクを開き、`お名前`を別の架空名へ再編集して保存する。
- [ ] 「保存しました」が表示され、再読込と管理画面の両方で変更後の値を確認する。
- [ ] あやこの LINE にはこの埋め込み回答の通知が届いていないことを確認する。

## 3. 停止・失効・改ざん耐性

- [ ] 通知を OFF にして保存し、使い捨ての埋め込み回答をもう 1 件送っても新しいメールが届かないことを確認する。
- [ ] 通知を ON に戻し、「発行済みの編集リンクを失効」を実行する。失効前の LINE/メール編集リンクが「このリンクは使用できません」になることを確認する。
- [ ] 新しく発行した編集リンクの末尾を 1 文字だけ変えた URL も拒否されることを確認する。改変 URL 自体は証跡へ残さない。
- [ ] 期限切れは lane の自動テストで確認済みとし、30 日待つための本番放置はしない。

## 4. 片付けと証跡

- [ ] 通知を OFF にし、使い捨てフォームと回答を承認済みの host 手順で削除する。
- [ ] 本番3フォーム、Formaloo、他の友だち、他のメール宛先に変更・送信がないことを再確認する。
- [ ] 下の実施記録だけを更新する。秘密値、PII、編集リンク全文、通知本文全文は書かない。
