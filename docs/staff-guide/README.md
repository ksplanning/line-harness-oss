# スタッフ用 使い方ガイド (staff-guide)

この中の `*.md` は、管理画面 (かんりがめん) の使い方を **スタッフ向けにやさしい日本語** で書いた資料です。
この資料は、管理画面の右下に出る **常駐ヘルプチャット (「使い方を聞く」ボタン)** の答えのもとになります。

- 迷ったら、管理画面のどのページでも右下の「使い方を聞く」を押して、日本語で質問してください。
- チャットは、この資料に書いてあることだけを根拠にお答えします。書いていないことは「資料にありません」と正直にお返しします (推測でウソを作りません)。

## この資料の取り込み方 (管理者向け)

資料を追加・更新したら、管理者が次のコマンドで取り込みます (取り込むと右下チャットが新しい内容で答えられるようになります)。

```
WORKER_URL=<管理画面のサーバー> API_KEY=<管理キー> node scripts/seed-staff-docs.mjs
```

- ファイル名がそのまま資料の「見分け名」になります (同じ名前で何度取り込んでも重複しません)。
- 各ファイルの先頭の `# 見出し` が資料タイトルになり、チャットの「根拠にした資料」に表示されます。

## 章立て (この順で読むと分かりやすい)

| ファイル | 内容 |
|---|---|
| `friends-management.md` | 友だち管理 (一覧・検索・タグ・対応マーク) |
| `broadcast.md` | 一斉配信 (全員やタグで絞った人へ一度に送る) |
| `broadcast-combo.md` | 組み合わせ配信 (1回の配信で複数の吹き出しをまとめて送る) |
| `template-packs.md` | 配信セット (テンプレパック / よく使う文面のセット) |
| `scenarios.md` | ステップ配信 (シナリオ配信 / 順番に自動で送る) |

> まだ書けていない章 (タグの絞り込み活用・高機能フォーム・よくある質問の設定・シート連携 など) は
> `docs/BACKLOG.md` に「これから書く章」として控えてあります。
